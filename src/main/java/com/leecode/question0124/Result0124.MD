给定一个非空二叉树，返回其最大路径和。

本题中，路径被定义为一条从树中任意节点出发，沿父节点-子节点连接，达到任意节点的序列。该路径至少包含一个节点，且不一定经过根节点。

 

示例 1：
```
输入：[1,2,3]

       1
      / \
     2   3

输出：6
```
示例 2：
```
输入：[-10,9,20,null,null,15,7]

   -10
   / \
  9  20
    /  \
   15   7

输出：42
```
----
思路
--
由于这次求的是路径，所以对任意一个节点，其三条路径只能选两条，因此对每个节点需要保存能够与上层节点链接的路径最大值和不能和上层链接的路径最大值  
对于不过父节点的最大值，不能加最高位置节点到当前节点中的值。

----
标准答案
--
由于每个节点的最大路径包括两种情况：自己为转折节点、自己作为左右子树中的一员。
给每个节点设置一个"贡献"，含义为这个节点在被上层节点计算时，能贡献出的最大的权重，对于一个节点，其贡献为节点的权重+max（左子节点贡献，右子节点贡献）
其最大路径和为权重+左子节点贡献+右子节点贡献     
所以最终的解法为：递归计算贡献，并在每个节点计算时算出以该节点为顶点的最大路径和，并记录最大值。
